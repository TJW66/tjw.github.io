计算机的基础知识（硬件）

发表于2025-11-30|更新于2025-11-30
|总字数:999|阅读时长:3分钟|浏览量:

笔记本中，cpu、gpu 通常是和主板集成在一起的。

cpu

型号：主流为锐龙（AMD）和英特尔（Intel）。（解析以笔记本为主，台式机不一样）

  1. AMD 的型号命名规则：R7-7745HX
    • R7：性能等级。数字越大越好。
    • 7：发布代数。
    • 7：市场定位。越大定位越高。
    • 4：结构代码。0，3都代表 zen3，其他数字一一对应（4->zen4）。
    • 5：架构分类。0 为标准架构，5 为升级版。
    • H：功耗代码。U 低压，H 标压。功耗越高性能越强，越耗电。
    • X：特殊标记。X 表示可超频；s 代表略微降低了功耗。
  2. Intel 的型号命名规则：i7-14650HX
    • i7：性能等级。数字越大越好。
    • 14：发布代数。
    • 650：性能代码。等级和代数接近的情况下，越大越好。
    • H：功耗代码。U 低压，H 标压。功耗越高性能越强，越耗电。
    • X：特殊标记。X 表示可超频。（只有 X）

核心：cpu 最主要的组成部分，cpu 所有的计算、接受/存储命令、处理数据都由核心执行。一个 cpu 有多个核心，核心的数量决定了 cpu 的性能水平。

线程：由核心延申出的虚拟执行单元，可以理解为每个核心具体去处理数据的数据流，早期一个核心对应一个线程，现在主流 cpu 基本都是一个核心对应双线程。

cpu 的频率：

  1. 频率定义：cpu 的频率分为基准频率（主频）和加速频率（睿频），单位 Ghz。主频可以认为是 cpu 在日常使用时的理论标准频率，睿频可以认为是 cpu 在执行高性能任务时的理论最高频率。理论频率越高，一般 cpu 的性能越强。
  2. 超频：利用 bios 的设定，增强 cpu 的供电水平来实现频率短暂超过睿频，性能可以得到小幅度增强，但长期使用会影响 cpu 寿命。

内存

内存也叫主存，负责临时存储 cpu 需要计算的数据，对笔记本性能有较大影响，内存损坏 cpu 无法正常工作。

内存容量：单位为 GB，主流笔记本内存为 16GB。容量越大笔记本性能越强。

内存频率：单位为 Mhz，主流笔记本频率为4800、5600、6000。内存频率越高内存性能越好。

注意：内存条可以进行更换，但是前提是频率需要相同。

硬盘

固态硬盘可以永久性存储数据。同时内存和 cpu 会从硬盘中读取数据，所以硬盘的性能对电脑有影响。

硬盘容量：单位为 GB。1GB=1024MB，1MB=1024KB，1KB=1024B

Pcie 标准：目前主流的硬盘都是采用 Pcie 标准，多为3.0和4.0，少数为5.0。数字越大硬盘性能越好。

读写速度：单位为 MB/S，读取大于写入。3.0硬盘一般在1000-3000，4.0的在3000-7000，5.0的可以上万。

GPU

型号：主要分为独显和核显。

  1. 核显：集成在 CPU 中，型号由 cpu 决定。cpu 性能越强 gpu 性能越强。
  2. 独显：基本上笔记本用的都是 NVIDIA（英伟达）独立显卡。常见三个系列：MX、GTX、RTX。
  3. 总结：RTX》GTX》MX，数字越大性能越好。

显示芯片：是显卡的核心部件，显示芯片的性能直接决定了显卡的性能上限。

显存：显卡的存储器，用于临时存储需要显示芯片处理的数据。

显存参数：主流显卡为GDDR。

电脑开机过程

  1. 按下开机键
  2. 供电给主板
  3. CPU 启动、散热启动
  4. 内存启动、屏幕亮起背光、bios 自检
  5. 屏幕出现主板品牌 logo、GPU 输出信号、从第一启动项（通常为硬盘）加载操作系统
  6. 系统各个重要组件和自启软件启动
  7. 验证用户密码，进入桌面
